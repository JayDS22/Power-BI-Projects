## Overview

This repository serves as a centralized portfolio of Power BI dashboard projects, each built to demonstrate proficiency in data manipulation, DAX calculations, data modeling, and visual storytelling. The dashboards span multiple industries and use cases, reflecting real-world business questions and analytical approaches.

**What you will find in each project folder:**

- `.pbix` file — the Power BI Desktop dashboard (interactive)
- Source data files (CSV / Excel) where applicable
- Supporting scripts (Python / R) used for data preprocessing

## Projects

### Finance and Investment

| Project | Description | Key Metrics |
|---------|-------------|-------------|
| **Bank Loan Insights** | Analyzes loan approval trends, borrower demographics, risk segmentation by income bracket, and default rate patterns across loan categories. | Approval rate, default rate, avg. loan amount, debt-to-income ratio |
| **Crypto Market Trends** | Tracks cryptocurrency market capitalization, price volatility, and comparative performance across major coins over time. | Market cap, 24h volume, price change %, dominance index |
| **Mutual Fund Analysis** | Evaluates mutual fund performance by category, NAV trends, risk-adjusted returns, and expense ratio comparisons. | NAV, CAGR, Sharpe ratio, expense ratio, AUM |

### Retail and Sales

| Project | Description | Key Metrics |
|---------|-------------|-------------|
| **Sales Performance Analysis** | Revenue tracking with regional breakdowns, product-level profitability analysis, and time-series trend identification. | Total revenue, profit margin, YoY growth, top products |
| **Superstore Retail Analytics** | Deep-dive into category performance, shipping efficiency, and the relationship between discounting strategies and profit erosion. | Sales by category, discount impact, shipping cost ratio |
| **Walmart Superstore Analysis** | Store-level performance benchmarking, seasonal demand patterns, and inventory movement insights. | Store-level sales, seasonal index, avg. basket size |

## Getting Started

**Prerequisites:** [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free download from Microsoft).

```bash
# 1. Clone the repository
git clone https://github.com/JayDS22/Power-BI-Projects.git

# 2. Navigate into the repo
cd Power-BI-Projects

# 3. Open any project folder and launch the .pbix file in Power BI Desktop
```

If a project includes source data files, they are located within the same project folder. Power BI will attempt to resolve data source paths on first load — update the source path in Power Query Editor if prompted.
